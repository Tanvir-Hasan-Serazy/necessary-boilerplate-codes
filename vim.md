# Vim Shortcuts

```
h -> Navigate Left
j -> Navigate Down
k -> Navigate Up
l -> Navigate Right
dd -> Delete Current Line
b -> Jump from word to word backward
w -> Jump from word to word forward
diw -> Delete the word on the cursor
yiw -> Copy the word on cursor
de -> delete current string to end of the string
db -> delete current string to start of the string
gg -> navigate to Start of the file
ggVG -> Select the whole file
shift'+ y -> Copy to register
$ -> End of the line
^ -> Start of the line
/ -> search forward search n or N o navigate
:%s/foo/bar/g -> Replaces all occurrences of "foo" with "bar" throughout the entire file ("%" for all lines).
:noh -> To turn off highlighting until the next search
shift v -> to select the whole line
```
